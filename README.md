# wallet_test
# Requirement
nodejs
# Install and Run
`npm install` `node main.js` running at port 8000

# Config database
edit config in `database/db.js`

# API
import `wallet_test.postman_collection.json` or [Postman link](https://www.getpostman.com/collections/7505c811ce4a93ea2f6f)
# Endpoint
`api.aofserver.site`
